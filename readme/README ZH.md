[b]是男人就下一百层![/b]

本模组提供了一个全新的可选模式 将正常游戏模式的圣山主线扩增至了40000深度!
一共有七个圣山和最后的实验室用于调整状态 在这之后等着你的是....被大幅强化的三眼
本模式的所有生物群系都将按难度以近乎疯狂的程度蔓延并打乱 在主线就可以体验到法师巢穴 冻结避难所 发电站 甚至...天空和地狱
不用担心游戏体验问题 开局将赠送随时编辑法杖等天赋和几把法杖
大部分特征地形依旧保留 你可以像往常一样往右拿到石板 甚至可以再一路往左获取水洞的血上限与完全回复生命
你可以设置自定义按键来显示简易地图 可不要迷路了
特别提醒 如果你种过太阳 请在最后一层探索时格外小心 因为它就在那里!

- 对部分法术的效果进行了调整
- 地图上生成的瓶子将更坚硬并装有各种奇怪的材料 包括炼金原液和贪婪诅咒液体 反应也更加多元
- 调整了部分天赋 天赋的说明也进行了相应的调整
- 调整了法杖等级和怪物分布 一些怪物将变得更加强大
- 完全生命回复现在不再额外增加10生命上限
- 部分法术和天赋将使用右键作为额外的操作键
- 更多改动等你探索 包括被重做的boss和NG+ 甚至...NG-!

[spoiler][b]-> 击杀BOSS提示:[/b]

龙(现在在第三层圣山下面的真菌洞穴右侧)的行动能力得到了巨幅加强 最有效的击杀方式是使用纯粹的投射物 切割和穿凿伤害
战利品包括轻盈/沉重一击等极为有用的修正法术和十级否杖等

三眼会给与对BOSS RUSH而言非常关键的法术, 包括白洞和小传送魔弹等

高阶炼金术士最有效的击杀方式是仅使用爆炸类型的伤害
击败它将获得点金免疫 这对击杀带有点金伤害类型的三眼赛旦很有帮助

携带高阶炼金术士掉落的音乐钥匙可以防止魔杖鉴赏家复活 而魔杖核心可以防止你被缴械
魔杖鉴赏家会尝试消除大部分的电浆和具有范围伤害的投射物

三眼赛旦会快速翻倍自己的血量 所以请速战速决
它的战利品包括顶级天赋死里逃生

叠加大量的电和暴击可以迅速蒸发机械三眼
如果你不能秒杀它 使用鹿灵掉落的毁灭法术可以快速处理它召唤的护盾无人机
击败它将获得源源不断的浮空能量 可以让你更好得击杀会窃取浮空能量的魔杖鉴赏家

三眼之足的所有攻击都会穿墙 并且它的自我克隆速度超乎你的想象
击杀它会随机掉落虚弱诅咒类法术 这是与遗忘者和法师领主战斗的关键

你需要至少摆放三只邪眼或利用究极火花的爆炸粒子才能让遗忘者显形 三眼赛旦的战利品就包括三只邪眼
击败它将获得火焰免疫和一个类似冰冻领域的效果 你可以到铁砧将这一效果重铸为纯粹的火焰免疫天赋
逃离遗忘者的惩罚非常惨重 所以请在迎战它前做好准备

鹿灵几乎只能用治疗伤害击杀(正常的治疗魔弹会对他造成伤害) 你最好一击毙命(最有效的方法是叠加暴击率)
击败它将获得变形免疫(这会使遁入虚空无效)

法师领主会无效化你的天赋 就像无效化祭坛所做的那样
秒杀不了它 就准备好拾取关键天赋等待时机 不然更恐怖的攻击将会袭来

最后 击杀boss的同时使用遁入虚空将会使得部分奖励失效 比如炼金给予的点金免疫
[/spoiler]
